# bamazon
bamazon version 1.0 10/26/2018


## Installation
1. Clone the git repository - `git clone https://github.com/sharon-f-lewis/bamazon.git`
1. cd bamazon
1. npm install

## General Usage Notes
Command line program to allow users to place orders from inventory.

`node bamazonCustomer.js`
